# 🎯 行銷 Prompt 生成器

一個專為行銷人員設計的 AI Prompt 生成工具，幫助您更有效地與 ChatGPT、Claude 等 AI 工具溝通，快速產出高品質的行銷內容。

## ✨ 功能特色

- **多種行銷目標支援**
  - ✍️ 文案撰寫
  - 📱 社群媒體貼文
  - 📢 廣告創意
  - 📧 Email 行銷
  - 🔍 SEO 內容
  - 💡 其他行銷需求

- **客製化選項**
  - 目標受眾設定
  - 4 種語調風格（專業正式、親切友善、創意活潑、緊迫感）
  - 3 種內容長度（簡短、中等、詳細）
  - 自訂特殊要求

- **使用者友善介面**
  - 響應式設計，支援手機、平板、桌面裝置
  - 視覺化選項選擇
  - 一鍵複製生成的 Prompt
  - 即時預覽結果

## 🚀 使用方式

### 方法一：直接開啟（推薦）

1. 下載本專案的所有檔案
2. 直接開啟 `index.html` 檔案
3. 無需任何伺服器或安裝步驟，瀏覽器即可執行

### 方法二：本地伺服器

如果您想在本地伺服器環境中運行：

```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Node.js http-server
npx http-server -p 8000
```

然後在瀏覽器開啟 `http://localhost:8000`

## 📖 操作步驟

1. **選擇行銷目標** - 點擊您想要創作的內容類型
2. **描述產品/服務** - 詳細說明您的產品或服務特色
3. **設定目標受眾** - 描述您的目標客戶群
4. **選擇語調風格** - 選擇最適合的溝通語氣
5. **選擇內容長度** - 根據需求選擇適當的字數範圍
6. **填寫特殊要求**（選填）- 加入任何額外的需求或限制
7. **點擊生成按鈕** - 系統會立即產生專業的 Prompt
8. **複製 Prompt** - 將生成的 Prompt 複製到 AI 工具中使用

## 💡 使用範例

### 範例一：社群媒體貼文

**輸入：**
- 行銷目標：社群媒體
- 產品描述：一款針對上班族的健康便當訂閱服務，每日新鮮製作，營養均衡
- 目標受眾：25-40歲的上班族，注重健康飲食
- 語調：親切友善
- 內容長度：簡短

**生成的 Prompt 會包含：**
- 角色設定
- 任務描述
- 產品資訊
- 目標受眾分析
- 社群媒體特定要求（hashtag、emoji、互動設計等）

### 範例二：廣告文案

**輸入：**
- 行銷目標：廣告創意
- 產品描述：企業級雲端儲存解決方案，提供無限空間和進階安全防護
- 目標受眾：中小企業 IT 主管
- 語調：專業正式
- 內容長度：中等

**生成的 Prompt 會包含：**
- 專業的廣告結構
- 主標題和副標題建議
- A/B 測試版本
- CTA 按鈕文字

## 🛠️ 技術架構

- **HTML5** - 語意化標記結構
- **Tailwind CSS** - 現代化 UI 設計框架（透過 CDN）
- **Vanilla JavaScript** - 原生 JS，無需額外框架
- **響應式設計** - 支援各種螢幕尺寸

## 📁 專案結構

```
prompt_gen/
├── index.html       # 主要 HTML 頁面
├── script.js        # JavaScript 邏輯
└── README.md        # 專案說明文件
```

## 🎨 自訂樣式

如果您想要修改顏色或樣式，可以在 `index.html` 的 `<script>` 標籤中找到 Tailwind 設定：

```javascript
tailwind.config = {
    theme: {
        extend: {
            colors: {
                primary: '#6366f1',    // 主要顏色
                secondary: '#8b5cf6',  // 次要顏色
                accent: '#ec4899'      // 強調顏色
            }
        }
    }
}
```
